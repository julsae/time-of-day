For Module 8, we started the class project, Time of Day. In class, we created four boolean if else statement so that the page would display a certain image and background color depending on the time of day. When it's morning, the background color will be orange and a certain image will display. When it's afternoon, the background color will be green. When evening, pink will be displayed, and lastly purple will be the background color you see when you access the page at night.
